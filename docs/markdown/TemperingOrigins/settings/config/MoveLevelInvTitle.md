## 移星界面标题

### 参数范围

| 配置项 | 免费版 | 付费版 | 定制版 |
|-----|-----|-----|-----|
| 本身  | 字符串 | 字符串 | 字符串 |

### 作用范围

| 游戏版本 | 插件版本 |
|------|------|
| 所有   | 所有   |

**默认值: "淬炼移星"** \
该项不可为空且建议不要过于简单越复杂越好

```yaml line-numbers
MoveLevelInvTitle: "淬炼移星"
```

如果觉得黑的好看也就是默认效果建议改为这样子的

```yaml line-numbers
MoveLevelInvTitle: "§d§b§c§d§m§r§0淬炼移星"
```

这样子可以避免在一些还没更新1.1.1的服务器出现的bug
